### 3. Verify the Installation
After installation, you can verify it by running:
```bash
jupyter notebook
```
or
```bash
jupyter lab
```

### 4. (Optional) Install Additional Kernels
If you want to add support for other programming languages, such as R or Julia, you'll need to install their respective kernels. For Python, the default kernel is installed automatically.

### 5. Running JupyterLab and Notebook
To run JupyterLab, use:
```bash
jupyter lab
```
For Jupyter Notebook:
```bash
jupyter notebook
```
Both commands will open the corresponding interface in your web browser.

### Troubleshooting
If you encounter any issues during installation, you might want to try alternative package managers like `apt` or `aptitude`. For more detailed guidance, refer to the official [Jupyter documentation](https://jupyter.org/install).

This method provides a straightforward way to install Jupyter on Kali Linux using the command line interface.
